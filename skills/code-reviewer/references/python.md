# Python 审查检查清单

## 1. 惯用 Python（Pythonic 代码）
- [ ] **PEP 8：** 代码是否遵循 PEP 8 风格指南（命名、缩进、空白）？
- [ ] **推导式：** 是否在适当的地方使用列表/字典/集合推导式替代简单循环？
- [ ] **上下文管理器：** 资源管理（文件、网络连接）是否使用 `with` 语句？
- [ ] **真值测试：** 是否惯用地使用真值测试（如 `if items:` 而不是 `if len(items) > 0:`）？

## 2. 类型提示与文档
- [ ] **类型提示：** 函数签名是否使用类型提示注解（PEP 484）？
- [ ] **文档字符串：** 模块、类和函数是否有描述性文档字符串（PEP 257）？

## 3. 性能与效率
- [ ] **生成器：** 处理大型数据集时是否使用生成器以节省内存？
- [ ] **内置函数：** 是否有效使用内置函数和标准库（如 `itertools`、`collections`）？

## 4. 异常处理
- [ ] **异常：** 是否捕获特定异常？避免 `except Exception:`
- [ ] **日志：** 是否使用 `logging` 模块而不是 `print()`？

## 5. AsyncIO
- [ ] **Await/Async：** 异步代码中是否等待所有 I/O 绑定调用？async def 内是否有阻塞代码？
